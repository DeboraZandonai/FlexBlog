<h2>
  💡 Como rodar o projeto?
</h2>

### Clone o repositório para ter uma cópia do código em sua máquina

```bash
$ git clone https://github.com/DeboraZandonai/FlexBlog.git
```

### Navegue até a pasta do projeto

```bash
$ cd FlexBlog
```
